# apeiner-bot

> Telegram chatbot for sending stats about trending NFT collections

## Getting started

### Prerequisites

- Node version 18

### Setup

```bash
git clone git@github.com:zsevic/apeiner-bot.git
cd apeiner-bot
cp .env.sample .env # change values after copying
nvm use 18
npm i
npm run dev
```

### Linting

```bash
npm run lint
npm run lint:fix
```

### Testing

```bash
npm test
```

### Technologies used

- Node.js, Express, Cron, Bottender
